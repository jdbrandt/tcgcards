PROJECT TITLE: Pokemon Trading Card Game Cards
PURPOSE OF PROJECT: Use the Pokemon Trading Card Game Specificaction in combination with the Pokemon Trading Card Game Game and Pokemon Trading Card Game Player classes to create a reasonable facsimile of the Pokemon Trading Card Game for simulatory purposes.
AUTHORS: Jack Brandt

-Designed to provide a backend for simulations of the Pokemon Trading Card Game
-Utilizes the various OO mechanisms (encapsulation, inheritance, etc.) we have learned throughout the year.
